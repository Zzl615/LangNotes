## 🏁 **学习目标**

| 项目        | 周期  | 达成标志                            |
| --------- | --- | ------------------------------- |
| **短期目标**  | 每月  | 词汇、听力、输出成果有记录  
| **阶段性目标** | 每季度 | 新能力显著突破（Q1习惯、Q2阅读、Q3表达、Q4整合）    |                |
| **长期目标**  | 全年  | 能用英语理解并表达技术内容（达到 CEFR B2–C1 水平） |
